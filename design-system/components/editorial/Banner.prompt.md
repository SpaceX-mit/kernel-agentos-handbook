Reversed tag box (feature-flag style) — the loudest small element in the system; use to tag a section, status, or catalog item.

```jsx
<Banner>MAGAZINE FOR CITY CODERS</Banner>
<Banner variant="kraft">SECURITY</Banner>
<Banner variant="ink">SHIP</Banner>
```

`variant`: `tomato` (default, loud), `kraft` (warm/quiet), `ink` (dark/quiet). Always uppercase content.
