The wire-style news dispatch feature tool — use when an issue reacts to a specific event, filed the night it happened. Repeating slug marquee, newspaper dateline, rotated gummed dossier card, checkbox-numbered propositions, optional BULLETIN, "— 30 —" terminator.

```jsx
<DispatchSpread
  slug="WIRE · COMPUTE-USE SHIPS · 速報"
  title="The desktop agent ships"
  titleJp="デスクトップ制御、出荷"
  deck="Filed before the takes industrialised."
  dateline="TOKYO — APRIL 12, 04:00"
  status="SHIPPED"
  filedAt="04:00 JST"
  propositions={[
    { n: '01', overline: 'WHAT HAPPENED', title: 'It opened at eight, before the coffee', body: ['And read the calendar unasked.'] },
  ]}
  bulletin="The restraint is the feature."
  terminator="end of dispatch · kernel.chat wire"
  signoff="Filed from a fifth-floor walk-up."
  issue={{ number: '376', month: 'APRIL', year: '2026' }}
/>
```
