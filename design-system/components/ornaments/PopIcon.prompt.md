Editorial pictogram primitive — hand-tuned 1.75px strokes, deliberately not Feather/Lucide. Use sparingly as section markers / sign-offs, never as UI buttons.

```jsx
<PopIcon name="asterisk" size="sm" color="var(--pop-tomato)" />
<PopIcon name="arrow" aria-label="continue" />
<PopIcon name="quote" size="lg" />
```

Names: `arrow · asterisk · sparkle · leaf · coffee · sun · moon · book · pin · quote · thread · pilcrow`. The **asterisk** is the one systemwide glyph (every folio strip) — the rest are single-use accents.
