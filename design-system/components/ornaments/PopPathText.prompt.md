Text-on-a-path ornament — curved / arched / waved short headlines in italic EB Garamond. Use for cover ornaments, section transitions, and sign-offs; never for body prose.

```jsx
<PopPathText text="Summer is Here" preset="arc-top" color="tomato" />
<PopPathText text="filed at dawn" preset="wave" size="lg" />
```

Presets: `arc-top` (rides up & over, stamp), `arc-bottom` (scoops, smile), `wave` (lopes). Pass a custom `d` in the 0 0 400 120 box for bespoke curves.
