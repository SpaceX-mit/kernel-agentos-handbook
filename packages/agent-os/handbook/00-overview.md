## 快速参考：关键发现

| 维度 | 核心发现 |
|------|----------|
| **语言选择** | TypeScript + Node 22，零外部依赖，追求可移植性 |
| **类型系统** | 判别联合类型替代异常，OSResult<T> 是核心模式 |
| **安全模型** | CVE 驱动设计，污染追踪是对 EchoLeak 的直接响应 |
| **命名哲学** | OS 类比驱动，POSIX 概念到 Agent 概念的映射 |
| **开源策略** | 双仓库 + Apache 2.0，OS 层免费 + 企业特性商业化 |
| **演进路径** | 竞品驱动 (CMA)，每次发布从竞品获取设计灵感 |
| **知识编码** | 文档即设计，README 是主要设计文档而非代码 |
| **团队心智** | 学术严谨 + 工程务实，引用层级分明 |
